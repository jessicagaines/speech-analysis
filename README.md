# speech-analysis

Scripts used to analyze acoustic speech data produced during altered auditory feedback experiments.
Written for Matlab R2024b.

This repository contains data and code used in the following manuscripts. Please cite the corresponding manuscript if using the data.

Gaines JL, Dale CL, Houde JF and Nagarajan SS (2025) Adaptation response to pitch shift during speech is attenuated upon relearning. Front. Hum. Neurosci. 19:1689023. doi: 10.3389/fnhum.2025.1689023
